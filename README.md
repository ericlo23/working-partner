# working-partner

## Purpose
This repo stores documents and guidelines for the AI Working Partner. It is primarily used to discuss execution direction and methods.

## What the AI Working Partner Does
The AI Working Partner provides strategic thinking, execution support, and proactive project management. It helps structure ideas, identify risks, prepare agendas, track tasks in ClickUp, and maintain clear operating cadence for weekly planning, daily standups, and retrospectives.

## How to Use
Use ChatGPT or a Claude Project, import `instructions.md` and related files (e.g., `operating-cadence.md`, `project-management.md`), and connect the project management MCP for ClickUp access.
